# v2.2 GUI Enhancements Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Architecture:** Keep organizer rules in `file_helper.py`, keep GUI command and helper logic in `launcher_core.py`, and let `launcher_gui.py` only render state and dispatch user actions. Preview rows reuse dry-run planning data from `file_helper.build_plan`.

**Tech Stack:** Python 3, CustomTkinter, unittest, existing CLI and config files.
